## Solution MVP (1-2 semaines)

### Features Core
1. **Upload DPE** (PDF parsing)
   - Extract: numéro ADEME, date, diagnostiqueur, notes
   - Parsing: pdfjs-dist + regex

2. **Vérification Basique** (Freemium)
   - Numéro ADEME valide ?
   - Diagnostiqueur certifié + référencé ADEME ?
   - DPE expiré ?
   - Format cohérent ?

3. **Red Flags Detection** (IA simple)
   - Notes anomalies (E/F gonflé vs surface/âge)
   - Dates inconsistantes
   - Diagnostiqueur warning list

4. **Expert Audit** (Premium)
   - Robin vérifie manuellement
   - Rapport détaillé PDF
   - CHF 200-500

5. **Pricing**
   - Freemium: vérif basique + flagging gratuit
   - EUR 15: rapport détaillé (recommandations)
   - EUR 500: expert audit Robin
   - B2B: EUR 50/mois agents immobiliers + API bulk

## Stack Technique

- **Frontend:** React + Next.js (PWA)
- **Backend:** Node.js + Express + PostgreSQL
- **PDF Parsing:** pdfjs-dist + regex
- **ADEME DB:** Scrape depuis data.gouv.fr (public)
- **Red Flags:** Heuristiques ML simples (pattern matching)
- **Bookings:** Calendly API
